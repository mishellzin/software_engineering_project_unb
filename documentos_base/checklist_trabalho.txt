- associação: expositor -> lista produto

Entidades Solicitadas:
- feira
	- nome -- OK
	- descrição -- OK
	- data inicio -- OK
	- data termino -- OK
	- local -- OK
	- cidade -- OK
	- estado -- OK
- expositor
	- nome -- OK
	- descrição -- OK
	- contato -- OK
- produto
	- nome -- OK
	- descrição -- OK
	- preco -- OK
- ingresso
	- nome de feira -- OK
	- número de ingresso -- OK
	- data -- OK

Como usuário, faço:
- listar feira
- listar expositor em feira
- listar produto de expositor em feira
- ler dados feira
- ler dados expositor em feira
- ler dados produtos de expositor em feira

- criar conta (USUÁRIO)
- autenticar-se (USUÁRIO)

- criar feira (ORGANIZADOR)
- editar feira (ORGANIZADOR)
- excluir feira (ORGANIZADOR)

- criar expositor (ORGANIZADOR/COLABORADOR)
- editar expositor (ORGANIZADOR/COLABORADOR)
- excluir expositor (ORGANIZADOR/COLABORADOR)

- criar lista de produto (EXPOSITOR)
- editar lista de produto (EXPOSITOR)
- excluir lista de produto (EXPOSITOR)

(EXPOSITOR)
- criar produto (EXPOSITOR)
- editar produto (EXPOSITOR)
- excluir produto (EXPOSITOR)

()
- criar ingresso
- ler ingresso	--	INGRESSO NÃO PODE SER EDITADO
- excluir ingresso

- ingresso permite acesso de feira na data de emissão

Regras impostas
- registro de feira, expositor, produto ou ingresso só pode ser editado ou excluído por quem o criou
- exclusão de registro só pode ser realizado se não houver registro associado a ele
